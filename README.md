# Roulette REST API

Casino game which accpets REST API request and responds in JSON format


## Tech Stack

**Server:** Apache Tomcat

**Database:** MySQL

**Framework:** Maven, Jersey, Hibernate

**Dependency** lombok, jersey-media-json-jackson
